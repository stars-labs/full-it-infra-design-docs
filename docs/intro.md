# 项目简介

本项目旨在为企业提供一套**现代化、开源、可落地**的 IT 基础架构设计方案。

为了方便阅读与协作，我们将文档体系重新架构为以下三个核心板块。无论您是 IT 管理者、架构师还是运维工程师，都能快速找到所需内容。

## 📚 文档导航

### [1. 核心设计 (Core Design)](./01-core-design/it-infra-design.md)
> **顶层规划与软件灵魂**
> 
> 这里定义了企业的 IT 战略方向。
> *   **[整体架构设计](./01-core-design/it-infra-design.md)**: 身份认证 (IAM)、MDM 与 ERP 的集成蓝图。
> *   **[开源软件栈](./01-core-design/oss-software-stack.md)**: 涵盖办公协作、DevOps、监控等全领域的最佳开源选型。

### [2. 硬件选型 (Hardware)](./02-hardware/server-selection.md)
> **物理基础设施基石**
> 
> 为您的数据中心选择最合适的物理设备。
> *   **计算**: 服务器配置与选型指南。
> *   **网络**: 核心/接入交换机与路由器推荐。
> *   **存储**: NAS 与 SAN 存储硬件方案。
> *   **机房**: 电力、冷却与机柜规划。

### [3. 架构方案 (Architecture)](./03-architecture/network-architecture.md)
> **技术实施与落地**
> 
> 具体的实施细节与运维策略。
> *   **[网络架构](./03-architecture/network-architecture.md)**: VLAN 划分、VPN 组网与路由策略。
> *   **[安全架构](./03-architecture/security-architecture.md)**: 零信任模型、防火墙策略与合规性。
> *   **[存储与备份](./03-architecture/storage-solution.md)**: 数据生命周期管理与灾备演练。
