## 采购注意事项

### 1. 售后服务
- **响应时间**: 确保4小时内响应,24小时内上门
- **备件供应**: 查询本地备件库情况
- **服务合同**: 建议购买3年维保服务

### 2. 预交付测试
- **功能测试**: 验证所有GPU和硬件组件正常
- **性能测试**: 运 行实际工作负载测试
- **兼容性测试**: 验证所选模型和软件栈

### 3. 扩展性考虑
- **GPU插槽**: 预留未来扩展空间
- **电源冗余**: 确保电源有30%以上冗余
- **网络带宽**: 考虑未来25G/100G升级

### 4. 采购时机
- **季度末/年末**: 通常有折扣促销
- **新品发布后**: 老款产品可能降价
- **批量采购**: 2台以上可谈折扣
