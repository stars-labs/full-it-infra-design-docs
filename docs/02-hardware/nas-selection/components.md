# 扩展配件与硬盘

## 扩展配件

### LSI 9211-8i HBA卡（IT模式）

用于扩展 SATA/SAS 接口，ZFS 直通必选。

| 规格 | 参数 |
|------|------|
| 接口 | 8× SATA/SAS (SFF-8087) |
| 速率 | 6Gbps |
| 模式 | IT模式（直通） |
| 价格 | ¥85-150 |

:::warning 购买注意
确认卖家已刷好 IT 固件（P20版本），否则需要自行刷机。IR 模式（RAID模式）不适用于 ZFS。
:::

### 内存（DDR5 SO-DIMM）

MinIO 推荐 DDR5 4800MHz，兼容性好且稳定。

**推荐配置：**
- **家用MinIO：** 16GB 足够（4个副本约消耗2-4GB）
- **生产环境：** 32GB 更稳妥

| 企业级 | 希捷银河 (Exos) | 8-16TB | ¥1200-2500 |

:::tip CMR vs SMR
ZFS 必须使用 CMR（传统磁记录）硬盘，**避免 SMR（叠瓦式）硬盘**。购买时确认型号是否为 CMR。
:::
