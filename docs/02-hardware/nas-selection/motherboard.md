# 主板与 CPU 选型

## 主板选型

### 畅网 N100/N305 六盘位 NAS主板

| 规格 | 参数 |
|------|------|
| **CPU集成** | N100 (4核) / N305 (8核) **焊死在主板上** |
| SATA | 6× SATA 3.0 (JMB585 扩展) |
| 网口 | 4× Intel i226-V 2.5GbE |
| 内存 | 1× DDR5 SO-DIMM (最大 32GB) |
| M.2 | 2× NVMe PCIe 3.0 x1 |
| 板型 | ITX 17×17cm |
| 供电 | DC 12-19V 宽压 |
| 功耗 | 待机 ~10W |

:::note 重要说明
**畅网 NAS 主板是集成CPU设计**，N100/N305 处理器已焊死在主板上，**无需单独购买CPU**。这是针对 NAS 场景的省电设计。
:::

**推荐选择：**
- **家用MinIO：** N305版本（8核更适合多并发）
- **低功耗场景：** N100版本（待机仅6-8W）

**官方支持：**
- 官网直购：https://changwang.com/product.html?id=46

| SATA | 8× SATA 3.0 |
| 网口 | 2× 2.5GbE + 1× 10GbE 万兆 |
| 内存 | DDR5 |
| 价格 | ¥1380-1440 |

**淘宝搜索：** `畅网 N355 八盘位 万兆 NAS主板`

:::note 重要说明
**畅网所有NAS主板都是集成CPU设计**，处理器已焊死在主板上，**无需单独购买CPU**。这种设计：
- 降低功耗
- 减少故障点  
- 适合7×24小时NAS运行
:::

## CPU 类型对比

| 主板系列 | CPU类型 | 是否集成 | 适用场景 |
|----------|---------|----------|----------|
| N100/N305/N355 | Intel Alder Lake-N | ✅ 已集成 | 家用/中小型NAS |
| W680 | Intel Core 12/13代 | ❌ 需另购 | 企业级/高性能NAS |
| X570D4U | AMD EPYC/Threadripper | ❌ 需另购 | 专业服务器 |

:::tip ZFS 与 ECC 内存
N100/N305 主板不支持 ECC 内存，家用场景问题不大。如需 ECC 支持（企业级 ZFS），请选择 W680 或 ASRock Rack X570D4U 等主板。
:::
