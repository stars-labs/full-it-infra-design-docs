# NAS 硬件选型采购方案

## 设计目标

- **支持 ZFS 文件系统**：适配 TrueNAS、飞牛OS 等 NAS 系统
- **5盘位以上**：满足 RAID-Z1/Z2 冗余需求
- **热插拔支持**：便于硬盘维护更换
- **低功耗运行**：7×24 小时运行，控制电费成本
- **扩展能力**：预留万兆网络、HBA 扩展空间

## ZFS 配置建议

### RAID-Z 方案

| 盘位 | 推荐方案 | 可用容量 | 容错 |
|------|----------|----------|------|
| 5盘 | RAID-Z1 (4+1) | 80% | 1盘 |
| 6盘 | RAID-Z2 (4+2) | 67% | 2盘 |
| 8盘 | RAID-Z2 (6+2) | 75% | 2盘 |
| 10盘 | 2× RAID-Z1 (4+1) | 80% | 每组1盘 |

### 硬盘选择

| 类型 | 推荐型号 | 容量 | 价格 |
|------|----------|------|------|
| NAS专用 | 西数红盘 Plus (WD Red Plus) | 4-8TB | ¥600-1200 |
| NAS专用 | 希捷酷狼 (IronWolf) | 4-8TB | ¥600-1200 |
| 企业级 | 西数金盘 (WD Gold) | 8-16TB | ¥1500-3000 |
| 企业级 | 希捷银河 (Exos) | 8-16TB | ¥1200-2500 |

:::tip CMR vs SMR
ZFS 必须使用 CMR（传统磁记录）硬盘，**避免 SMR（叠瓦式）硬盘**。购买时确认型号是否为 CMR。
:::

## 验收测试

### 硬件检测

- [ ] CPU 温度正常（<70°C）
- [ ] 内存识别完整
- [ ] 所有 SATA 接口识别
- [ ] 网口 Link 状态正常
- [ ] 热插拔功能测试

### ZFS 测试

- [ ] 创建存储池成功
- [ ] scrub 扫描通过
- [ ] 快照功能正常
- [ ] SMB/NFS 共享可用

### 性能基准

- [ ] 顺序读写 > 200MB/s（机械盘）
- [ ] 网络传输接近网口带宽
- [ ] SMART 状态健康
